# MSSE642 Assignment 2 — Submission Instructions

## What’s Included

| File | Purpose |
| ------ | --------- |

| `HikingClub_Architecture_Diagram.png` | Architecture diagram for Part 2A |
| `architecture-diagram.mmd` | Mermaid source (for edits/export) |

## How to Create Your Word Document

1. **Open** `Assignment2_HikingClub_ThreatModel.md` in any text editor or Markdown viewer.

2. **Copy** the content into Microsoft Word (or Google Docs).

3. **Insert** the architecture diagram:
   - In Word: **Insert → Pictures → This Device**
   - Select `HikingClub_Architecture_Diagram.png`
   - Place it in **Part 2A** where the diagram is described

4. **Optional:** Use [Mermaid Live Editor](https://mermaid.live) if you want to change the diagram:
   - Open `architecture-diagram.mmd` and copy its contents
   - Paste into Mermaid Live Editor
   - Export as PNG and replace the image in your document

5. **Save** the document as:

   ```text
   LastnameFirstname_Assignment2.doc
   ```

   (or `.docx` — the assignment specifies `.doc` for compatibility)

6. **Proofread** for correctness and academic integrity. Adjust wording to match your style.
